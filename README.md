Контент загружается только с белого списка доменов (в файле secure_hosts.json), защищая вас от утечек ip адреса, список обновляется раз в 12 часов с гитхаба.

На ВКВидеоЛайв присутствует прокрутка чата при загрузке медиа. На кике тоже. Твич как-то по особому с этим работает, не удалось добавить. Также 7тв переделывает чат, поэтому там работает через fallbackLoop() так как c 7tv не удалось прикрепить MutationObserver. С 7tv проверено работает, с остальными расширениями не известно.

Буду рад коментариям к сложности работы кода.
