Контент загружается либо с белого списка доменов (в файле secure_hosts.json), защищая вас от утечек ip адреса, либо вообще все ссылки содержащие медиа-расширения в конце (.jpg .mp4 и.тд) список обновляется раз в 12 часов с гитхаба. На недоверительных доменах с включеной защитой есть кнопка с возмозностью загрузки.

На ВКВидеоЛайв присутствует прокрутка чата при загрузке медиа. На кике тоже. Твич как-то по особому с этим работает, не удалось добавить. С 7tv проверено работает, с остальными расширениями не известно.

Буду рад коментариям к сложности работы кода.

Как установить? 
Перейдите в релизы https://github.com/msvxdlss/twitch-images-in-chat/releases
скачайте zip архив, распакуйте в удобное место на диске, в хроме включите Режим разработчика в расширениях, и укажите папку с этими файлами, всё.
